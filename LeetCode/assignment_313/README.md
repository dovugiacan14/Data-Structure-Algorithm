# Assignment 313: Super Ugly Number 

A super ugly number is a positive integer whose prime factors are in the array primes 

Given an integer n and an array of integers primes, return the $n^{th}$ super ugly number 

The $n^{th}$ super ugly number is guaranted to fit in a 32-bit signed integer. 

Example 1: 
- Input: n = 12, primes = [2, 7, 13, 19]
- Output: 32 
- Explanation: [1, 2, 4, 7, 8, 13, 14, 16, 19, 26, 28, 32]

Example 2: 
- Input: n = 1, primes = [2, 3, 5]
- Output: 1 
- Explanation: 1 has no primes factors, therefore all of its prime factors are in the array primes = [2, 3, 5]
