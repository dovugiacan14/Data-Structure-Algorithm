# Assignment 321: Create Maximum Number 

You are given two integer arrays ```nums1``` and ```num2``` of   lengths m and n respectively. ```nums1``` and ```num2``` represent the digits of two numbers. You are also given an integer k.
Create the maximum number of length ```k <= m + n``` from digits of the two numbers. The relative order of the digits from the same array must be preserved.

Example 1: 
- Input: nums1 = [3, 4, 6, 5], nums2 = [9, 1, 2, 5, 8, 3], k = 5 
- Output: [9,8,6,5,3] 

Example 2: 
- Input: nums1 = [3,9], nums2 = [8,9], k = 3
- Output: [9, 8, 9]
